This is a very simple filter utilizing teh ChemDoodle web component to show
JCAMP-DX file (*.jdx) in the ChemDoodle web component.

TO INSTALL:

1. Unpack this .zip or .tar.gz download.
2. Copy the /chemdoodle folder (and all its contents) inside the /filter folder of your Moodle 2.0.x installation i.e. alongside the folders for other filters.
3. Activate the filter in Moodle's filter admin screen.
